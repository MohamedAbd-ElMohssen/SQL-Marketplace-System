# SQL Marketplace System

## 📌 Overview
This project simulates a real-world marketplace system using SQL.

It includes database design, data insertion, and advanced analytical queries.

## 🧱 Database Design
The system includes:
- Customers
- Sellers
- Products
- Orders
- Payments
- Reviews

## ⚙️ Features
- Complex JOIN operations
- Aggregation queries
- Subqueries and EXISTS
- Window Functions (RANK, Running Total)
- Data Quality validation

## 📊 Example Insights
- Top selling products
- Customer spending analysis
- Revenue per seller
- Monthly best customers

## 🧪 Data Quality Checks
- Detect mismatched order totals
- Detect invalid payments

## 🛠️ Technologies
- SQL Server (T-SQL)

## 🚀 How to Run
1. Run schema.sql
2. Run insert_data.sql
3. Run queries.sql

## 🎯 Skills Demonstrated
SQL, Data Modeling, Data Analysis, Data Quality
